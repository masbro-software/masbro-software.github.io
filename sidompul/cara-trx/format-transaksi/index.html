Format Transaksi

Transaksi Pulsa

Format transaksi: pulsa#DENOM.NO-HP@TRX-ID.MAX-HARGA. Nilai TRX-ID dan MAX-HARGA bisa tidak digunakan, Contoh:

pulsa#5000.08111111@123
pulsa#1000.08111111
pulsa#25000.08111111
pulsa#5000.08111111@123.5200

Contoh untuk otomax: pulsa#50000.[tujuan]@[trxid].

Transaksi Paket Data

Format transaksi: paket#KODE-PRODUK.NO-HP@TRX-ID.MAX-HARGA. Nilai TRX-ID dan MAX-HARGA bisa tidak digunakan, Contoh:

paket#NA571.08111111
paket#NBR1.08111111
paket#NBR1.08111111@1234.20000

Contoh untuk otomax: paket#KODE-PRODUK.[tujuan]@[trxid]. Default status adalah Sukses (trx ini masih menunggu respons balasan dari server, cek SMS/ Inbox untuk detail status trx)

Darimana asal KODE-PRODUK ? Cek melalui menu Tab Tools -> Produk, bagian product_code.

Transaksi Transfer Kuota

Kode voucher/ perdana disimpan di addon

Format transaksi: transfer-kuota#besar-kuota.NO-HP@TRX-ID. Nilai TRX-ID bisa tidak digunakan. Contoh:

transfer-kuota#3,5G.08111111@123
transfer-kuota#6G.08111111
transfer-kuota#11G.08111111
transfer-kuota#21G.08111111
transfer-kuota#31G.08111111

transfer-kuota-hybrid#3,5G.08111111
transfer-kuota-hybrid#6G.08111111
transfer-kuota-hybrid#11G.08111111
transfer-kuota-hybrid#21G.08111111
transfer-kuota-hybrid#31G.08111111

Kode voucher/ perdana TIDAK disimpan di addon (di kirim dari server pulsa)

Format transaksi voucher biasa: transfer-kuota-langsung#KODE_VOUCHER.NO-HP@TRX-ID.

Format transaksi voucher hybrid: transfer-kuota-langsung#KODE_VOUCHER#NOMER_PERDANA.NO-HP@TRX-ID. Nilai TRX-ID bisa tidak digunakan. Contoh:

transfer-kuota-langsung#KODE_VOUCHER.08111111
transfer-kuota-hybrid-langsung#KODE_VOUCHER#NOMER_PERDANA.08111111
