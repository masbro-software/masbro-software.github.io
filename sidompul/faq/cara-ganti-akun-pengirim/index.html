Cara Ganti Akun Pengirim

1. Buka menu SETTING (pastikan proses TRX tidak kondisi berjalan). Buka tab UMUM

2. Ganti dengan akun pengirim baru pada nomor 2

3. Klik tombol SIMPAN

4. Buka PIDGIN, kemudian buka menu Tools -> Plugins

5. Pilih menu __XMPP Gateway__SIDOMPUl sesuai gambar:

6. Klik tombol Configure di sebelah bawah jendela:

7. Ganti dengan akun pengirim yang baru:

8. Klik tombol Close

Pastikan kedua akun penerima dan pengirim sudah menjadi teman/ Friend. Untuk caranya, sesuaikan dengan software server pulsa anda.
