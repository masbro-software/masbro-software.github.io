RefID: 29386,paket#00017568.082XXXX.20200, TRX TERLALU LAMA, ANTRIAN DIJALANKAN LAGI,11/04/2020 20:19:08

Pesan ini berarti software tidak mendapat response selama +- 60 detik. Bisa dikarenakan jaringan internet yang tidak stabil/ server digipos gangguan/ sibuk. Untuk pesan dengan tipe ini, anda harus cek manual status laporan pulsa/ paket untuk nomor dan produk tersebut. Untuk proses ulang/ batalkan/ sukseskan buka tab TRX, klik kanan data yang akan diulang kemudian pilih aksi yang diinginkan. Trx tidak bisa ditembak ulang sebelum dibatalkan manual.
